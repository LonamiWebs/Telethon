GetFileHashesRequest

Both users and bots can use this request. See code examples.

---functions---
upload.getFileHashes#9156982a location:InputFileLocation offset:long = Vector<FileHash>

Returns

A list of the following type is returned.

FileHash

This type can only be an instance of:

FileHash

Parameters

locationInputFileLocation
offsetlong

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.upload.GetFileHashesRequest(
        location=types.InputFileLocation(
            volume_id=-12398745604826,
            local_id=42,
            secret=-12398745604826,
            file_reference=b'arbitrary\x7f data \xfa here'
        ),
        offset=-12398745604826
    ))
    for x in result:
        print(x)
